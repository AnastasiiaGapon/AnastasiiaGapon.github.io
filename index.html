About

I’m a self-taught graphic/web designer from Kharkiv Ukraine. Very excited about the new features of HTML5, such as the new semantic tags, and the power of using elegant CSS decisions instead of heavy loaded JS on a page.

I study JS and PHP in the background.

I understand the significance of well-organized HTML and CSS performance for producing efficient output. The mandatory using preprocessors and naming conventions in my workflow reduces the specificity level and makes the final code understandable and reusable not only for myself.
